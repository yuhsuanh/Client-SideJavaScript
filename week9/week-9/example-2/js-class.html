JavaScript - Now With Classes!

EcmaScript 2015 introduce classes to JavaScript that allow easier and cleaner syntax that more closely resembles other programming languages. However, at it's core, JS still relies on prototypal inheritance to get things done.

Inheritance With Classes

We can create a subclass (subclassing) in order to allow objects to inherit other objects To do so, we use the extends keywords. When using subclass, however, 'this' isn't automatically initialized for a class defined by the extends keyword. This initialization is depedent on the parent class constructor (the constructor functon of the class from which you are extending). We can use the super operator to intialize 'this' by calling the parent constructor.
