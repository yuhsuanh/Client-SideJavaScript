The Layers of a Website/Application

We can think of a website or application as a cake. The first layer is the HTML (content), the second layer is CSS (presentation) and the third layer is JavaScript (interactivity)

To take our cake analogy even further, we can think of HTML as the cake (content), CSS as the icing (presentation) and JavaScript as the birthday candles (interactivity).
