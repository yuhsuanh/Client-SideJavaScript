Conditional Statements & Decision Making

When creating dyanmic web applicatons, we often need to evaluate or test a condition and then invoke certain code based on the result. For example, if the day of the week is Saturday, display a customized message that says 'Happy Weekend!' to our user.

We can use if/else statements or switch statements to evaluate and then run certain code in response.
