Arrays

Arrays allow us to store a list or sequence of related data. The items we store inside of an array are sometimes called elements. Elements can be any type of data (number, string, boolean, objects). We can even store arrays inside of arrays!.

There are two techniques avaiable in JavaScript that allow us to create arrays. We can create array literals or we can use an array constructor. It's generally best to stick with the array literal, however, if you are creating a large, empty array, the array constructor can come in handy.
