Loops

In programming, we often need to repeat the same thing over and over again. To save time (and code!), we can use loops to help us achieve this.

There are a few different types of loops we can use, depending on our functionality needs. A for loop is often the most commonly used loop.

Open your console to check out the JavaScript examples. Remember, you can also write JavaScript dirently in your browser console.
