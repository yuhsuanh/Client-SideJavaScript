Object Literals

In JS, most things are objects. We can create our own objects to encapsulate related functions (methods) and variables (properties).

We call objects that we create object literals, they are different than objects instantiated from a class. Object literals help us create efficient ways to group, store & transmit our data. Objects can contain any type of data (strings, booleans, arrays and even objects). This data is stored in object members that consist of name-value pairs.

Let's dive deeper into Object Literal syntax, how to access object members, how to update object members and how to add new object members. We'll use cats to help us out (are you surprised?).
